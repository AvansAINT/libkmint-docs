The mainloop of a kmint application.

main_loop does the following:

  1. It determines the time that passed since the last tick
  2. It calls function f with the time delta and a reference to a kmint::loop_controls object
  3. If f sets kmint::loop_controls::quit to true, the simulation is terminated immediately
  4. Otherwise
    1. If kmint::loop_controls::pause is false, stage s will have its act method called.
    2. And then, if kmint::loop_controls::render is true, all actors are rendered
  5. Go back to step 1
Parameters
sthe stage which contains the actors
wthe window in which to render the stage
fa callable object that accepts a delta_time kmint::loop_controls object.
