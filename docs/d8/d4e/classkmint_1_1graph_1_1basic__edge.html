Detailed Description

template<typename NodeInfo>
class kmint::graph::basic_edge< NodeInfo >

Represents a weighted directed edge.

This class represents a directed edge in a node. Directed edges have a direction. That is, they can be seen as a one way street from a starting point (

See also
basic_edge::from) to a destination (
basic_edge::to).

basic_edge maintains references to its starting point and destination and consequently, does not own any nodes.
