Constructs a map by obtaining ownership of a graph.

Because map obtains the ownership of a map_graph, the caller must pass in an rvalue reference. For example:

#include <utility> // includes std::move
kmint::map::map_graph g {};
// ...
map m { std::move(g), kmint::math::isize{1280,720} };
