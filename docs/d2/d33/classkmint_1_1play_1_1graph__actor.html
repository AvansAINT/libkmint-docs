Detailed Description

template<typename Graph>
class kmint::play::graph_actor< Graph >

A actor that is used to draw a graph on screen.

A graph actor should not interact with its environment. Its primary purpose is to return a drawable that draws the graph

Template Parameters
Graphthe type of graph this actor represents
