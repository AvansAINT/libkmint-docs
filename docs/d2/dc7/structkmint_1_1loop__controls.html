Detailed Description

Controls various aspects of the simulation loop.

A reference to a loop_settings object is given to the callable object given to kmint::main_loop. The behaviour of the simulation can be controlled by changing the member variables.

Member Data Documentation

◆ pause

bool kmint::loop_controls::pause {false}

If true, the stage will not be updated.

This effectively pauses the simulation. The current state will be maintained.

◆ quit

bool kmint::loop_controls::quit {false}

If set to true, the simulation will end.

In the event that quit is set to true by the main function, the simulation will terminate immediately.

◆ time_scale

double kmint::loop_controls::time_scale {1.0}

Controls the factor by which time is scaled.

For a value of 1.0 the game time equals wall time. For a value less than 1 the game will run at a slower pace, for a value larger than 1 the game will run at a faster pace.
